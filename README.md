# South Korea Top 50 Spotify Analysis

## Project Overview

This project analyzes the South Korea Top 50 Spotify Playlist to study:

- Comeback Momentum
- Chart Re-entry
- Fandom Engagement
- Artist Performance
- Album Analysis

An interactive Streamlit dashboard was developed to visualize the insights.
